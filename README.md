# 📝 To-Do List App (Python)

## 📌 Overview
A simple command-line To-Do List application to manage daily tasks.

---

## ✨ Features
- Add tasks  
- View tasks  
- Delete tasks  
- Saves tasks in a file  

---

## 🛠️ Tech Used
